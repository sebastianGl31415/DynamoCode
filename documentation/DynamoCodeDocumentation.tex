% >> ifprint
% Funktion: Zwischen Druck- und Digitalversion umschalten
% Hinweise: Die Digitalversion ist für den Gebrauch am Rechner gedacht. Aus
%           diesem Grund werden Verweise auf Abschnitte etc. farbig
%           hervorgehoben. Die Druckversion besitzt nur schwarze Schriften.
%           Bei der Digitalversion wird zudem der Textkörper mittig auf der
%           Einzelseite zentriert, was beim doppelseitigem Druck durch das
%           Zusammenwirken von zwei Seiten nicht der Fall ist.
% Gebrauch: Entweder \printtrue oder \printfalse
% Referenz: -

\printtrue % Druckversion
